## 📊 Understanding the results

The software explains performance metrics in plain language. You will see scores for your pages. A high score means your site works well. A low score indicates needed repairs.

### Largest Contentful Paint
This measure tracks how long your largest image or block of text takes to load. If this takes a long time, users see a blank page. The tool identifies large files that you should resize or compress.

### Cumulative Layout Shift
This measure tracks how much content moves around while a page loads. A page that jumps around confuses users. The tool finds elements without defined sizes that cause this movement.

### Interaction to Next Paint
This measure tracks how fast your site reacts when a user clicks a button. High numbers mean the site feels sluggish. The tool tells you which processes interfere with user clicks.

## 🛠 Fixing common issues

You do not need to be a programmer to benefit from this tool. When the audit finishes, the software provides a list of tasks. 

- Image Optimization: The tool detects images that exceed the necessary size. It provides the links to these files so you can swap them for smaller versions.
- Code Cleanup: The tool identifies scripts that block page rendering. You can pause or move these scripts to improve speed.
- Cache Settings: The tool reviews your server settings. It provides instructions to enable browser caching for faster repeat visits.
